## Installation

Installation is easy!

    $ gem install hexe

If you want to, you can add this line to your application's Gemfile:

    gem 'hexe'

And then execute:

    $ bundle install

to install it.

## Usage

It's really easy!

    $ hexe

If you installed it using Bundler, you can even do this:

    $ bundle exec hexe
